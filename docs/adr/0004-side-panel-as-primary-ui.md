# ADR 0004: Side Panel As Primary UI

## Status

Accepted.

## Context

The product needs room for capture status, memory review, source inspection, search, and copy/inject controls. A popup is too transient for this workflow.

## Decision

ContextVault MVP will use Chrome Side Panel as the primary browser UI.

## Consequences

Benefits:

- persistent workspace beside AI pages
- better review and search ergonomics
- natural place for recall suggestions

Tradeoffs:

- Chrome-focused MVP
- UI must handle narrow panel widths carefully
